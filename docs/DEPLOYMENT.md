# Deployment And GitHub Notes

## Repository Layout

Recommended public repository layout:

- `frontend/`
- `backend/`
- `docs/`
- `assets/`

## Local Run

### Frontend

Open:

- `frontend/index.html`

### Backend

Run:

```bash
cd backend
npm start
```

Default backend port:

- `4000`

## GitHub Pages

The frontend can be hosted as a static site through GitHub Pages.

If you use GitHub Pages:

- point it to the `frontend/` folder or publish a built static copy
- remember the map depends on external Leaflet/OpenStreetMap assets

## Suggested Repository Name

- `govgig-portal`
- `govgig-portal-prototype`

Government-backed gig worker registry portal prototype with company sourcing, compliance tracking, and health insurance marketplace support.
