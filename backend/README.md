# Backend

This folder contains a minimal mock backend scaffold for the GovGig Portal prototype.

## Purpose

It is not fully integrated with the frontend yet. It exists to provide a GitHub-ready backend structure and sample API shape.

## Endpoints

- `GET /health`
- `GET /api/workers`
- `GET /api/company-requests`
- `GET /api/insurance-plans`

## Run

```bash
cd backend
npm start
```

Default port: `4000`
